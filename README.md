Bank Queue Simulation System
Overview

This project simulates and compares different queue management strategies in a banking environment. The goal is to analyze how various queue structures impact efficiency, wait times, and overall system performance.

The simulation models real-world customer flow and evaluates multiple approaches to handling service lines.

Features

Implements and compares multiple queue types:

Standard queue (FIFO)

Array-based queue

Priority queue (for prioritized service handling)

Simulates customer arrival and service processes

Evaluates efficiency of single-line vs. alternative queue structures

Demonstrates differences in performance across data structures

Technologies Used

C++

Object-Oriented Programming

Custom data structures:

ArrayQueue

PriorityQueue

Standard Queue

Key Concepts Demonstrated

Data structures and algorithm design

Queue management and scheduling systems

Simulation of real-world systems

Performance comparison between different implementations

Circular array implementation for efficient memory usage

Example Implementation Detail

The array-based queue uses a circular buffer approach to efficiently manage memory and avoid unnecessary shifting of elements.
